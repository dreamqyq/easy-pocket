## 项目介绍
![](public/favicon.ico)

基于 React 技术栈实现的简易记账项目。
### 项目预览（手机扫码查看）
![](src/static/qrcode.png)
### V0.1.0
项目主要技术栈包括：React、TypeScript、LocalStorage、React-Router、React Hook、styled-component等。可实现基本的记账功能以及数据保存功能（数据保存基于 LocalStorage）。

## 项目运行命令

### 启动 
`yarn start`
### 构建
`yarn build`
### 部署
`yarn deploy`
